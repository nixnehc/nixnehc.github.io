LaTeX for Logcians: from Zero to Hero

逻辑学家的LaTeX：从入门到入土 （持续更新）

 Posted on April 18, 2023  |  3 minutes  |  497 words  |  Xin❤️

This is a crash course in LaTeX for myself (continuously updating).

Basic setting (short document)

\documentclass[UTF8, 11pt, a4paper]{article}
%加载自己定义的宏包，字体设置在这个包里
\usepackege{chenxin_chinese} 

%% 设置页边距：
\usepackage{geometry}
\geometry{a4paper,left=2.5cm,right=2.5cm,top=2cm,bottom=2cm}

%%%%%%%%%%%%%%%%%%%%%%%
\title{Notes on logic xxxxx}
\author{The Great xxxxx}
\date{} % 或者填入 \today

%%%%%%%%%%%%%%%%%%%%%%%%
\begin{document}
\maketitle  % 如果你想显示标题的话
%%%%%%%%%%%%%%%%%%%%%%%%%   以下是正文 %%%%%%%%%%%%%%%%
\tableofcontents % 生成目录

\section{this is a section xxx}

\subsection{this is a subsection xxx}

\subsubsection{this is a subsub....}

%%%%%%%%%%%%%% 参考文献页 （如果需要的话）
\newpage
\bibliographystyle{plain} % 参考文献风格
\bibliography{/Users/chenxin/Desktop/chenxin.bib} % bib文档地址
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\end{document}

Basic setting (long document)

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
%%    ChenXin's templat for long document      %%

\documentclass[UTF8,12pt,a4paper]{report}
\usepackage{chenxin_chinese} %% 加载自己定义的宏包

%% 设置页边距：
\usepackage{geometry}

\linespread{1.2}  %n倍行距

%% 设置全文脚注连续编号
\counterwithout*{footnote}{chapter}

% ####### 以下设置本文中常用的缩写 #########
%%此向下箭头适用于混合语言中的downarrow
\newcommand{\da}{\downarrow}
%%% 框架 和 模型
\newcommand{\F}{\mathfrak{F}}
\newcommand{\M}{\mathfrak{M}}
\newcommand{\T}{\mathfrak{T}}
\newcommand{\G}{\mathfrak{G}}

\newcommand{\h}{\mathcal{H}} % 混合语义 H
\newcommand{\nom}{\mathsf{NOM}} % 专名集
\newcommand{\Prop}{\mathsf{Prop}} % 命题变元集
\newcommand{\K}{\mathbf{K}} % 极小正规逻辑 K
\newcommand{\diff}{:\iffs} % 定义中的 iff 符号

%  rule
\newcommand{\der}{ \; / \;}  % der is derivation

%%   模型操作
\newcommand{\dju}{\biguplus_i} % 不交并
\newcommand{\gsm}{\rightarrowtail}  % 生成子模型
\newcommand{\bm}{\twoheadrightarrow}  % bounded morphic image
\newcommand{\meq}{\leftrightsquigarrow}  % modal equivalence

chenxin_chinese.sty 文件要和xxxx.tex文件放同一个文件夹。chenxin_chinese.sty 文件中的具体内容：

%%   chenxin_chinese.sty 文件具体内容
%%   本宏包在tex文件中的名称：
\NeedsTeXFormat{LaTeX2e}
\ProvidesPackage{chenxin}[2022/07/15 ChenXin's LaTeX package]

%%					1. 一些预置宏包
%%---------------------------------------------------------------
%% 各类数学宏包
\usepackage{amsmath}
\usepackage{amsthm}	% 注意 amsthm 最好在 amsmath 后面加载
\usepackage{amssymb}
\usepackage{mathtools}
\usepackage{amsfonts}
\usepackage{wasysym}  % 该包使得\Diamond，\Box比较好看，更敦厚些
\usepackage{stmaryrd} % \llbracket \rrbracket
\usepackage{mathrsfs} % 提供花体字母

%% 以下命令设置定理类环境的结束符号，本列中为 $\dashv$
\makeatletter
\def\@endtheorem{\hfill $ \dashv $\endtrivlist\@endpefalse }
\makeatother

%% 这两个忘记是干啥的啦。。。
\usepackage{enumerate}
\usepackage{framed}

%% 使用外部超链接
\usepackage{url} 

%% 交叉引用和参考文献
\usepackage{cite}
\usepackage{hyperref}
% 超链接文字的颜色设置：
\hypersetup{
	colorlinks=true,
	linkcolor=blue,
	filecolor=green,      
	urlcolor=brown,
	citecolor=cyan, }  

%%	图片设置
\usepackage{graphicx} 	% 插入图片的宏包
\usepackage{float} 		% 设置图片浮动位置的宏包
\usepackage{subfigure}	% 插入多图时用子图显示的宏包

\usepackage{times}  % 使用Times New Roman字体

%% 页面多栏；用于实现在同一页中实现不同的分栏
\usepackage{multicol}

%% 注脚宏包
\usepackage{footmisc}

%% 文本框
\usepackage{tcolorbox}
\tcbuselibrary{skins, breakable, theorems}


%%           	字体设定（超级麻烦！！）
%%-------------------------------------------------
%% 英文
\usepackage{fontspec}
\setmainfont[Mapping=text-text]{Times New Roman}
\setsansfont[Mapping=text-text]{Tahoma}
\setmonofont{Courier New}

%%中文
\usepackage[CJKchecksingle,CJKnumber]{xeCJK}
\setCJKmainfont{Songti SC}[ BoldFont = Songti SC Black , ItalicFont = Kai ]
\setCJKsansfont{Songti SC}
\setCJKmonofont{Songti SC}


%%    		  定义中文的数学环境样式
%%----------------------------------------------
%%  定理、定义、引理和推论等都统一连续编号，采用章号作前缀
\theoremstyle{remark}
%% \theoremstyle{style} 中， style 参数有3种：
			% plain：定理名称是正体，定理内容是斜体
			% definition：定理名称和内容都是正体
			% remark：定理名称是斜体，定理内容是正体
					
\newtheorem{thm}{\bf{定理}}[section] % 定理
\newtheorem{def.}[thm]{\bf{定义}} % 定义
\newtheorem{lemma}[thm]{\bf{引理}} % 引理
\newtheorem{corollary}[thm]{\bf{推论}} % 推论
\newtheorem{prop}[thm]{\bf{命题}} % 命题
\newtheorem{example}[thm]{\bf{例}} % 例子
\newtheorem{note}[thm]{\textit{注记}} % 注记


%%  将证毕符号(QED)改成黑色的正方形
\renewcommand{\qedsymbol}{$\blacksquare$}    


%%     一些数学环境中的逻辑符号，常用符号的自定义设置
%%----------------------------------------------------------------
%% 中文：当且将当 	\IFF
\newcommand{\IFF}{\text{ 当且仅当 }}

%% 数学环境中的 iff 符号：短的粗双箭头	  \iffs
\newcommand{\iffs }{\;\Leftrightarrow\;}

%% 正文环境中的iff符号：短的粗双箭头	\iffz
\newcommand{\iffz}{$ \iffs \; $}

%% if... then... 符号   \then
\newcommand{\then}{\text{\; $\Rightarrow$ \;}}

%% bisimulation 符号   \bisimilar
\newcommand{\bisimilar}{ \;{\raisebox{.3ex}[0mm][0mm]{\ensuremath{\medspace \underline{\! \leftrightarrow\!}\medspace}}} \;}

%% 实质蕴含, 虽然我都是用 \to 
\newcommand{\imp}{\rightarrow}

%% 一般的逻辑语言符号  
\newcommand{\lang}{ \mathcal{L} }

%%  数学环境中的破折号    \dd
\mathchardef\mhyphen="2D
\newcommand{\dd}{\mhyphen}


%% 中文文档类小结标题左对齐
% \CTEXsetup[format={\Large\bfseries}]{section}

	reflexive point/.default={25},
}
